## 🌟 Overview

Skillora is a full-stack AI-powered career preparation platform built using the MERN Stack. It empowers students and professionals to strengthen their resumes, optimize ATS compatibility, identify skill gaps, and prepare for interviews with AI-generated questions and intelligent feedback.

The platform combines modern web technologies with Artificial Intelligence to streamline the entire job preparation journey—from resume analysis and ATS scoring to personalized interview practice and career insights—all within a single intuitive application.
